# TFM MuBiB – Emma Reig Martí

This repository contains the work associated with the **Master’s Thesis Final Project** of the *Master's Degree in Bioinformatics and Biostatistics (UOC-UB)*, titled: 

**Analytical and Experimental Benchmarking of commercial Design of Experiments tools for Mixed Polymeric Micelles Optimization** 

The project focuses on an Exploratory Data Analysis (EDA) and pre-processing pipeline to optimize the formulation of Soluplus/TPGS Mixed Polymeric Micelles (MPMs) loaded with Docetaxel (DTX). It conducts a Design of Experiments (DoE) using different commercial DoE softwares, evaluating and comparing their predictive performance.
---

## Description & Objectives

The main objectives of this project are:

- Execute experimental designs using commercial DoEs tools to define the design space and optimize MPMs formulations.
- Perform an EDA using Python to identify patterns, detect errors and ensure the integrity of raw experimental data before applying to DoE software.
- Develop a Python-based standardized pipeline to handle Dynamic Light Scaterring (DLS) multimodality. This implements statistical similarity metrics (such as the *Crámer-von Mises* distance) to transform complex size distributions into scalar descriptors.
- Evaluate and compare the predictive accuracy of commercial DoE software.

## Main scripts

- `DLS_functions+metrics.ipynb`     → Contains the core functions and metrics used throughout the project for the calculate the similarity distance imp. 
- `DLS_data_analysis.ipynb`         → It uses the script 'Dls_functions+metrics.ipynb' to process the DLS raw data. 
- `Sampling_initial_points.ipynb`   → Plots the formulations proposed in each iteration during the experimental analysis
- `Synthesis_metrics.ipynb`         → Computes the Mean Absolute Error (MAE) of the experimental data generated by the Design-Expert software based on sampling points. 
- `Analysis_(software_name).ipynb`  → Performs the analytical evaluation for each sofware individually. It applies mathematical functions to asses performance.
